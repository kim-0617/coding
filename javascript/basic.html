자바스크립트 기본 규칙

  1. 하나의 실행문이 끝나면 마지막에 세미콜론 (;)을 붙여줍니다.
    세미콜론 (;)을 붙여 주지 않아도 상관은 없으나, 자바스크립트를 처음 학습하는 경우에는 하나의 실행문을 확인하고 넘어가는 차원에서 세미콜론 (;) 붙여서 코딩하는 것이 좋습니다.

    var num = 5;
    document.write(num); // 5
